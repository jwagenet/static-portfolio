Connect to U: drive on non-university computer

The U: drive provided by the university can be connected to while on campus from your own computer.

Connect to U: Drive on Windows

Placeholders {username} and {password} refer to your Pacific ID username and password respectively.

  1. Open 'Computer' from Start menu or desktop. (Winkey +E)
  2. Select 'Map network drive' from top menu.
  3. Select an available drive letter, such as 'U:'.
    Use the folder \\homedirs1.stk.pacific.edu\{username}
    Select 'Connect using different credentials' and Finish.
  4. Use stk\{username}' and {password} for your credentials.

The drive should now be located in 'Computer' when connected.

Connect to U: Drive on OSX

Placeholders {username} and {password} refer to your Pacific ID username and password respectively.

  1. Open Finder and navigat to the 'Go' menu and select 'Connect to Server...'
  2. Use the server smb://homedirs1.stk.pacific.edu/{username}.
  3. When prompted use stk/{username} and {password} as your credentials.
  4. Use stk/{username}' for the username and {password} for the password.

The drive should now be located on the desktop when connected.

Connect to U: Drive on Linux

Placeholders {username} and {password} refer to your Pacific ID username and password respectively.

  1. The U: drive can be connected directly via command line:
    mount.cifs //homedirs1.stk.pacific.edu/{username} /path/to/mount -o username={username},ntlm=sec

The drive should now be located on the desktop when connected.
